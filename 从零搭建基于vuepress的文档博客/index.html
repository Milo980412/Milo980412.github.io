从零搭建基于VuePress的文档博客

本文会帮助你从头搭建一个简单的 VuePress 文档，并且做一些简单的配置。

一、前提条件

VuePress 需要 Node.js >= 8.6

直接点击下载，或者在DOWNLOADS里按需下载即可

安装好后，查看nodejs版本

1

node

二、快速上手

1.创建并进入一个新目录

1

mkdir vuepress-milo && cd vuepress-milo

2.使用你喜欢的包管理器进行初始化

1

yarn init # npm init

初始化过程如无特殊需要，一路回车即可

3.将 VuePress 安装为本地依赖

不再推荐全局安装 VuePress

1

yarn add -D vuepress # npm install -D vuepress

4.创建docs文件夹，和一篇md文档

1

mkdir docs && echo '# Hello VuePress' > docs/README.md

5.在 package.json 中添加一些 scripts

这里推荐用vscode打开后编辑

1

code .

这一步骤是可选的，但我们推荐你完成它。

1
2
3
4
5
6

{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

6.在本地启动服务器

1

yarn docs:dev # npm run docs:dev

三、基本设置

此时的文档博客很丑，下一步我们需要做一些基本配置

1.目录结构设置

VuePress 遵循 “约定优于配置” 的原则，推荐的目录结构如下：

⚠️建议：先把这些都创建好，然后在再写入内容

2.public静态资源目录

docs/.vuepress/public: 静态资源目录，这里我放了图片作为页面主图和偏爱图标

3.重写默认颜色常量

docs/.vuepress/styles/palette.styl: 用于重写默认颜色常量，或者设置新的 stylus 颜色常量

默认是绿色，我这里更改为蓝色

默认页面有点窄，这里更改为900px宽度

1
2
3
4
5
6

// 颜色
$accentColor = #235DC8
// 页面设置
.page .theme-default-content:not(.custom){
    max-width: 900px!important;
}

docs/.vuepress/config.js: 配置文件的入口文件，也可以是 YML 或 toml，这里简单设置了网站标题、网站描述、偏爱图标。

module.exports = {
    // 网站标题
    // 网站描述
    // 偏爱图标
    // 导航栏与侧边栏设置
    themeConfig: {
        // logo设置
        logo: '/aniya.png',
        // 导航栏设置
        // 侧边栏设置
        sidebar: {
            // 周报
            '/pages/weekly/':
                [
                    {
                        title: 'Milo的学习周报',   // 必要的
                        collapsable: true, // 是否可折叠, 默认值是 true,
                        sidebarDepth: 2,    // 侧边栏深度, 默认值是1
                        children: [
                            'start',
                            '20220822',
                        ]
                    },
                ],
            // 推荐系统&&知识图谱
            '/pages/papers/':
                [
                    {
                        title: '推荐系统&&知识图谱',   // 必要的
                        collapsable: true, // 是否可折叠, 默认值是 true,
                        sidebarDepth: 2,    // 侧边栏深度, 默认值是 1
                        children: [
                            'rskg/start',
                            'rskg/DKN',
                            'rskg/RippleNet',
                            'rskg/MKR',
                            'rskg/KGCN',
